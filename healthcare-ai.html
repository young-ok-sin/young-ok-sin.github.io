README

의료 데이터 인증과 외부 연동을 함께 다룬 백엔드 프로젝트

의료 데이터 송수신, 외부 의료 API 연동, 블록체인 전송 로직, 원격 서버 운영까지 연결된 프로젝트였습니다. 팀장과 백엔드 개발을 동시에 맡으며 서비스 기능과 기술 구현을 모두 주도했습니다.

Role

What I owned

Service Goal

블록체인 기반의 의료 데이터 인증과 AI를 활용한 노약자 맞춤형 의료 애플리케이션을 개발해 데이터 신뢰성과 편의 기능을 함께 제공하는 것이 목표였습니다.

Ownership

  • 팀장으로서 일정과 개발 진행 조율
  • 백엔드 회원 관리 코드 개발
  • 외부 의료 API 통신 코드 개발
  • 의료 데이터 해시 변환 후 블록체인 전송 로직 구현

Implementation

Key features

01 Core Features

의료 데이터 송수신, 건강 편의 기능, 의료 질의응답 커뮤니티를 주요 기능으로 구성했습니다.

02 Backend Work

사용자 계정 관리와 외부 API 통신을 담당했고, 의료 데이터를 해시로 변환해 블록체인으로 전송하는 백엔드 로직을 구현했습니다.

03 Operations

원격 서버 운영을 맡아 개발 환경과 실행 환경을 유지했고, 서비스 흐름이 실제로 작동하도록 운영까지 연결했습니다.

04 Outcome

창의융합종합설계2 교과목에서 A+를 받았고, 이후 관련 프로젝트 경험이 대외활동과 후속 포트폴리오 구성의 기반이 됐습니다.
